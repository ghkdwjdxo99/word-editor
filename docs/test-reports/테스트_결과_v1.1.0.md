## V110-F01 자동 검증

- 한글, 영문, 숫자, 공백 포함 문자열 검색
- 검색 결과 없음과 빈 검색어
- 대소문자 구분 켜기/끄기
- 여러 결과의 다음/이전 이동
- 문서 처음·끝 순환
- 중첩 Inline 표시 본문 검색
- 문단 경계 비연결 정책
- 결과 선택 후 본문과 `IsDirty` 미변경

## WPF UI 확인

- Ctrl+F 입력 바인딩으로 찾기 패널 열림
- `찾기 Search 찾기 123`에서 두 결과의 다음 이동 및 `2/2` 상태 표시
- 찾기 전 선택을 패널 종료 시 복원
- 검색과 패널 종료 후 `IsDirty=False` 유지
- 검색 전후 본문 유지

## 남은 위험

1. 저장 차단 대상인 Table/List/Section 내부는 검색 대상이 아니다.
2. 정규식, 전체 결과 동시 강조, 여러 문서 검색은 기획 범위 밖이다.
3. IME 조합 중 실시간 검색과 고배율 화면 배치는 독립 테스터의 수동 확인이 필요하다.
4. 개발자 검증은 독립 테스터 승인과 v1.1.0 배포 승인을 대신하지 않는다.
