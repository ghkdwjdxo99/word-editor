## 남은 위험

1. 저장 차단 대상인 Table/List/Section 내부는 검색 대상이 아니다.
2. 정규식, 전체 결과 동시 강조, 여러 문서 검색은 기획 범위 밖이다.
3. IME 조합 중 실시간 검색과 고배율 화면 배치는 독립 테스터의 수동 확인이 필요하다.
4. 개발자 검증은 독립 테스터 승인과 v1.1.0 배포 승인을 대신하지 않는다.
