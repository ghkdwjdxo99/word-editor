# v1.0.0 개발자 작업 요청

상태: 검토 요청 — 구현 및 자동 테스트 통과, 테스터 승인 대기  
대상 패치: V100-P11  
대상 결함: BUG-V100-001  
우선순위: P0

## 개발자에게 전달할 메시지

간단 워드 편집기 v1.0.0의 다음 작업으로 V100-P11을 진행해 주세요. 설치 프로그램이나 새로운 편집 기능은 만들지 마세요.

현재 Table, List, Section 같은 비지원 블록이 있으면 저장 자체는 안전하게 차단되지만 오류가 `Unknown`으로 분류되어 사용자가 원인과 해결 방법을 알 수 없습니다.

다음 요구사항을 구현해 주세요.

1. `UnsupportedEditorContentException`을 미지원 콘텐츠 오류로 분류합니다.
2. 감지된 블록 종류를 사용자 메시지에 포함합니다.
3. 일반 텍스트로 다시 붙여넣거나 비지원 블록을 제거하라는 해결 방법을 안내합니다.
4. 기존 저장 차단과 원본 보호 동작은 유지합니다.
5. 현재 실패 중인 `V100-P02 비지원 편집 블록 사용자 안내` 테스트를 통과시킵니다.
6. 전체 자동 회귀 테스트를 실행합니다.
